# Band Name Generator

Simple web app that generates random band names based on user input. This is a study project, but the implementation and logic were written by me as part of my learning journey.

# About the project

This project was created as part of a learning process while studying backend development with Node.js and Express.

The main goal was to understand:

- Handling GET and POST requests
- Working with templates (EJS)
- Passing data between server and frontend

# Features

- Displays a welcome message on page load
- Generates a random band name when the button is clicked
- Shows the current year

# Tech Stack

- Node.js
- Express
- EJS
